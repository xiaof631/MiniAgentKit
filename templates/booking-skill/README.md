# Booking Skill Template

This template is generated by:

```bash
node packages/cli/dist/index.js create-skill booking --name booking-skill --output examples/booking-skill
```

It produces a WeChat mini program AI mode SKILL skeleton with:

- `SKILL.md`
- `mcp.json`
- `index.js`
- `apis/`
- `components/`
- `mini-agent.skill.json`

The checked-in `examples/booking-skill` directory is the current rendered version of this template.
